## 🚀 Instalación Rápida

### Opción 1: Shopify CLI (Recomendado)
```bash
cd theme
shopify theme dev
```

### Opción 2: Subir directamente a Shopify
1. En tu admin de Shopify, ve a Temas
2. Carga este repositorio como tema personalizado
3. Personaliza colores y configuración
